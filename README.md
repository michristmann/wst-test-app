# WST Test App

## Docker Setup

In the project directory, you can run the following command lines:

#### `Build with no cache`

```
docker-compose build --no-cache
```

#### `Start the services`

```
docker-compose up
```

#### `List the services`

```
docker-compose ps
```

#### `List the containers`

```
docker ps
```

#### `Stop services`

```
docker-compose stop
```

Once the Docker `services` are running, open [localhost](http://localhost:3000) to view it in the browser.
